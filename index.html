Mis Proyectos

01

Formulario interactivo

Formulario interactivo diseñado para mejorar la experiencia de registro con validaciones en tiempo real. Incluye verificación de campos requeridos, validación de email y teléfono, comprobación de coincidencia de contraseña.

 Además, cuenta con una barra de progreso dinámica que muestra el nivel de seguridad de la contraseña. A medida que se cumplen las condiciones, se marcan con un tick verde y se brinda retroalimentación positiva al usuario en el momento.

HTML5, CSS3, JavaScript, Bootstrap

Live Project Repositorio GitHub

02

Gestión de empleados app

Aplicación desarrollada para la administración de empleados de una organización.

 Permite agregar, actualizar y eliminar registros, además de realizar búsquedas avanzadas con filtros por salario, departamento y otros criterios.

 Integra una base de datos robusta y una interfaz gráfica intuitiva, facilitando la gestión de la información de forma ágil y eficiente.

Java (Spring Boot, Swing, Maven), MySQL, HTML5 y CSS3

Live Project Repositorio GitHub

03

Supermercado eccommerce

Simula una tienda online donde los usuarios pueden explorar productos, añadirlos al carrito y realizar compras de manera sencilla.

 Incluye un resumen de compra y un mensaje de confirmación con los detalles del pedido, lo que enriquece la experiencia de usuario.

HTML5, CSS3, JavaScript

Live Project Repositorio GitHub

04

API REST productos

Aplicación backend que implementa una API RESTful que expone endpoints como GET /users, POST /login, y operaciones CRUD completas (crear, leer, actualizar y eliminar productos).

 El servidor se encarga de estructurar las respuestas y gestionar la lógica de negocio, mientras que Maven facilita la gestión de dependencias.

 Aunque cuenta con una interfaz básica en Bootstrap, el objetivo principal del proyecto es la construcción de un servicio backend sólido y escalable.

Java (SpringBoot), MySQL, HTML5, CSS3 y Bootstrap

Live Project Repositorio GitHub

05

Web educativa interactiva

Proyecto web educativo que incorpora elementos multimedia como videos, audios y animaciones.

 Una de sus funciones más destacadas es el sistema de comentarios en tiempo real, que permite la interacción inmediata entre usuarios y fomenta la comunicación.

 Además, incluye cuestionarios interactivos y retroalimentación instantánea, creando un entorno dinámico.

HTML5, CSS3, JavaScript

Live Project Repositorio GitHub

06

Reservas restaurante

Aplicación web orientada a la gestión de reservas. Los usuarios pueden realizar reservas online con validaciones en tiempo real, recibiendo confirmaciones inmediatas y retroalimentación visual al completar el formulario.

 Además, incluye una vista administrativa que permite al personal del restaurante gestionar reservas, verificar disponibilidad y actualizar la información dinámicamente gracias al uso de AJAX, Fetch y WebSockets, lo que asegura una experiencia fluida y actualizada tanto para clientes como para administradores.

Node.js, Express, EJS, MySQL, HTML, CSS, JavaScript y Bootstra
